## 2. What LevelForge Is

LevelForge is a Discord bot used for XP, levels, rankings, leaderboards, and server-specific configuration.

## 3. Data We Process

Depending on feature usage, LevelForge may process and store the following data:

- Discord Server ID (`guildId`)
- Discord User ID (`userId`)
- Discord Channel ID (`channelId`)
- Discord Category ID (`categoryId`)
- XP values
- Levels
- Timestamps such as `last_xp_timestamp`
- Server configuration such as XP per message, cooldown, and level-up channel
- Ignored channels and ignored categories
- Plan or premium status such as `FREE`, `VIP`, `TESTER`, `LIFETIME`
- Admin actions such as XP changes by server admins

## 4. Data We Do Not Store

LevelForge does not intentionally store the following as part of normal bot operation:

- Passwords
- Payment data
- Private messages
- Permanent storage of full message contents
- Email addresses, unless a user contacts the operator directly

## 5. Purpose of Processing

Data is processed to:

- Provide XP and level functionality
- Calculate ranks and leaderboards
- Maintain server-specific configuration
- Prevent abuse (for example through cooldowns)
- Support administration, debugging, and support requests
